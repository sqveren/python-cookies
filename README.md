# Amazon Region & Cookie Handler

Цей проєкт вирішує проблему доступності товарів на Amazon для користувачів з України за допомогою маніпуляції кукі та налаштувань локалізації.

## 🚩 Проблема
Amazon автоматично визначає IP користувача і часто показує статус "Item not available" або "No delivery to Ukraine", що заважає автоматизації покупок.

## 💡 Рішення
1. **Force Language:** Встановлення `en-GB` через ChromeOptions.
2. **Cookie Persistence:** Збереження сесії у формат `pickle`, щоб не проходити перевірки щоразу.
3. **Region Spoofing:** Встановлення поштового індексу (ZIP-коду) Великої Британії для розблокування кнопок покупки.

## 📂 Структура проєкту
- `WorkWithCookies/` — основна логіка роботи з сесіями.
- `cookies/` — тимчасове сховище серіалізованих кукі (ігнорується Git).

## 🔧 Використання
Просто запустіть основний скрипт. Якщо кукі існують — вони будуть підтягнуті автоматично. Якщо ні — скрипт створить нову сесію.
